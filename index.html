Estilos en CSS
  1. Introducción a CSS

    2. Que es CSS?
    CSS(Cascade Style Sheets) Hojas de estilo en cascada, nos sirve para crear diversos estilos para nuestros elementos en HTML. Pudiendo definir de forma estructurada, la manera en la cual se organizarán estos estilos. CSS, nos permite ahorrarnos mucho trabajo, ya que al crear una plantilla de estilos que puede ser aplicacada rapídamente, evitamos tener que crear un estilo individual para cada elemento.

  2. Como usarlo

    3. Existen 3 formas de utilizar CSS

    • Externa
      • Para utilizar el código css de forma externa podemos utililizar un archivo diferente dedicado exclusivamente al código css.
      Para ello utilizamos la etiqueta
      link rel="stylesheet" href="miestilo.css"
    • Interna
      • También podemos usar el código CSS en nuestra página HTML usando la etiqueta:
      <style>
      body{ color: green; }
      </style>
